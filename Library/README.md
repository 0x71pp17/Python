## README: `library_system.py`

A simple library management system built with Python and object-oriented programming.

### Features
- **Book Class**: Represents a book with a title, author, and availability status.
- **Library Class**: Manages a collection of books.
- **Core Functions**: Add books, display all books, and search for a book by its title.
- **Checkout System**: Books can be checked out and returned, updating their availability.

### Prerequisites
- **Python 3.6 or later**

### Usage
Run the script:
```bash
python library_system.py
```

The script will:
1.  Create a `Library` instance.
2.  Create several `Book` instances.
3.  Add books to the library.
4.  Display information (title, author, availability) of all books in the library.
