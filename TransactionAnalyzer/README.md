## README: `transaction_analyzer.py`

A command-line tool to analyze bank transaction data.

### Features
- Prints a summary of all transactions, showing total deposits, withdrawals, and the final balance.
- Analyzes transactions to find the largest deposit and withdrawal.
- Calculates the average deposit and withdrawal amounts.
- Interactive menu to choose between printing a summary or performing an analysis.

### Prerequisites
- **Python 3.6 or later**

### Usage
Run the script:
```bash
python transaction_analyzer.py
```

1.  The program will display a menu with three options: `print`, `analyze`, or `stop`.
2.  Enter your choice when prompted.
3.  The program will perform the selected action and then return to the menu.
4.  Type `stop` to exit the program.
