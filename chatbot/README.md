## README: `cyber_init.py`

A cyberpunk-themed terminal chatbot written in Python using the **Rich** library for stylized output. Simulates a DEFCON-level system login by collecting user details and responding with themed messages.

### Features
- Hacker-style dialogue and prompts
- Colorful, formatted output via **Rich**
- Input validation with error handling
- Compatible with **macOS, Linux, and Windows**

### Prerequisites
- **Python 3.8 or later** (required by Rich)
- `rich` library

### Installation
```bash
pip install rich
```

### Usage
Run the script:
```bash
python cyber_init.py
```


### Alternative Script
If you prefer not to install `rich`, use `cyber_init_basic.py`. It provides the same interaction using standard `print()` and `input()` functions.
