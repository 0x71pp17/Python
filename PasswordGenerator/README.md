# Password Generator

A simple, secure Python script to generate cryptographically strong passwords using the `secrets` module.

## Features

- Generates random passwords with 17 characters by default
- Uses `secrets` for cryptographically secure randomness
- Includes uppercase, lowercase, digits, and special characters
- Easy to customize length and character set

## Prerequisites

- Python 3.6 or higher (recommended)

## Usage

1. Save the script as `password_generator.py`
2. Run the script using:

 ```bash
 python password_generator.py
 ```
