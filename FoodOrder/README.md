## README: `food_order_system.py`

A simple command-line Food Order System that lets users choose from different cuisines.

### Features
- Choose between Italian and Indian cuisine.
- View the available meals for the selected cuisine.
- Order a specific meal by name and quantity.
- Receive a summary of the order or a notification if the meal is not on the menu.

### Prerequisites
- **Python 3.6 or later**

### Usage
Run the script:
```bash
python food_order_system.py
```

1.  You will be greeted with a welcome message.
2.  Enter `Italian` or `Indian` to select your desired cuisine.
3.  The available meals for your chosen cuisine will be displayed.
4.  Enter the exact name of the meal you want to order.
5.  Enter the quantity you wish to order.
6.  The program will display your order summary (e.g., "2 x Margherita pizza") or "Meal not found" if the meal is not on the menu.
