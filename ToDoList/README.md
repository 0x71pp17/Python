## README: `todo_list.py`

A simple command-line To-Do List application written in Python.

### Features
- Add new tasks to your list.
- Remove the last task added.
- View the current list of tasks.
- Exit the program.

### Prerequisites
- **Python 3.6 or later**

### Usage
Run the script:
```bash
python todo_list.py
```

1.  The program will display your current task list (or a message if it's empty).
2.  Choose an option from the menu:
    *   **1) Add Task**: Enter the new task when prompted.
    *   **2) Remove Task**: Removes the last task you added. If the list is empty, it will notify you.
    *   **3) Quit**: Exits the application.

The program will continue to run, showing the updated list and menu, until you choose to quit.
